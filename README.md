# CRM-система

### Некоммерческий продукт 

Приложение развёрнуто на [Amvera Cloud](https://amvera.ru) — облаке для хостинга ботов, сайтов и других IT-приложений.  
База данных (PostgreSQL) развёрнута на [Supabase](https://supabase.com) - платформе разработки Postgres.

Приложение:  
https://crm-project-vofka.amvera.io  

Swagger:  
https://crm-project-vofka.amvera.io/swagger
